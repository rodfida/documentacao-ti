# Documentação de TI

Repositório com documentações, procedimentos e soluções de problemas comuns no ambiente corporativo de TI.

O objetivo é organizar conhecimento técnico e facilitar a resolução de incidentes no dia a dia.

---

## 📌 Criação de usuário (padrão corporativo)

### Passos:
1. Acessar o sistema de gestão de usuários
2. Criar novo usuário com:
   - Nome completo
   - Email corporativo
   - Departamento
3. Definir permissões conforme área
4. Gerar senha temporária
5. Solicitar troca no primeiro acesso

---
